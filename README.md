# HEADCIRCUS

## Project Overview

HEADCIRCUS is a platform designed to manage and visualize circus performances and artist interactions.

## Features

- **Performance Management:** Schedule and manage circus performances.
- **Artist Profiles:** Create detailed profiles for artists, including their skills and past performances.
- **Visualization Tools:** Visualize relationships between performers and performances.

## Installation

To install HEADCIRCUS, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/cotcollective/HEADCIRCUS.git
   ```
2. Navigate to the project directory:
   ```bash
   cd HEADCIRCUS
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage

To start the application, run:
```bash
npm start
```
